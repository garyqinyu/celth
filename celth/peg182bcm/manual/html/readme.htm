Installation

To install the remaining files on this disk, you will need to use either PKUNZIP or WinZip to extract the enclosed ZIP files to your computer. Unzip the files contained on each disk  into any root directory of your choosing. Please be sure to use the "-d" or "Use Folder Names" option when extracting the enclosed files, since the demonstration programs produce several sub-directories under the root installation folder.

Demonstration Files
Once you have extracted the zipped files, you will find the following sub-directories under your root installation folder:

\peg\utils\wb: This directory contains the WindowBuilder executable program, pwinbld.exe, along with a short tutorial that will help you begin using this powerful tool.

\peg\manual: This directory contains the PegX programming and reference manuals. You can browse any part of the documention by simply opening the file index.htm in your favorite browser.

\peg\examples: This directory contains several sub-folders. Each sub-folder contains a working PEG application program built to run under Windows. This is a standard environment supported by PEG, enabling you to do nearly all of your user interface development before your target hardware becomes available.

The example programs, while generally simple, are intended to demonstrate some basic features of PEG. The programs include the following:

\graphics
This program demonstrates the use of several PEG drawing primitives. The drawing primitives use various colors, line and border widths, and fill styles. This program also demonstrates overlapping, resizable windows.

\guage
This program demonstrates the use of PegBitmap and double-buffered drawing techniques. Using double-buffering allows smooth, flicker free animations.

\pegdemo
This program demonstrates many of the standard controls and windows available in the PEG class library.

\robot
This program demonstrates the use of PegImageConvert to produce an optimal color palette for the display of high-color depth images.
