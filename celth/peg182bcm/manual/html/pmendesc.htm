PegMenuDescription
 

PegMenuDescption is a structure used to define PegMenuButton objects. Arrays of PegMenuDescriptions are used to define any number of nested menus and sub-menus. PegMenuDescription arrays are terminated with an entry filled with 0 or NULL values. 

The PegMenuDescription structure is defined as: 


struct PegMenuDescription

{

    PEGCHAR *pText;

    WORD wId;

    WORD wStyle;

    PegMenuDescription *pSubMenu;

};
