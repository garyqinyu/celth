PegBitmapButton

Overview

PegBitmapButton is a PegButton class that displays a PegBitmap centered in the button client area.

 

See Also

PegButton

PegTextButton

PegRadioButton

PegCheckBox

Style Flags

PegBitmapButton supports the standard frame styles FF_NONE, FF_THIN, FF_THICK, FF_RAISED, and FF_RECESSED.

PegBitmapButton also supports the button styles BF_REPEAT and BF_DOWNACTION.

Signals
PegBitmapButton sends the PSF_CLICKED signal when selected.

Derivation
PegBitmapButton is derived from PegButton.
 
 

Constructors:

PegBitmapButton(PegRect &Rect, PegBitmap *bm, WORD wId = 0, WORD wStyle = AF_ENABLED)
PegBitmapButton(SIGNED iLeft, SIGNED iTop, PegBitmap *bm, WORD wId = 0, WORD wStyle = AF_ENABLED)
The first constructor creates a PegBitmapButton with a user defined size. The second constructor allows the button to automatically size itself relative to the bitmap size. In each case, the PegBitmap associated with the button will be displayed in the center of the button client area.

Public Functions:

virtual void Draw(void)
PegBitmapButton overrides the Draw() function to draw the associated bitmap.

virtual void SetBitmap(PegBitmap *nbm) { mpBitmap = nbm;}
This inline function assigns the bitmap associated with the button. The bitmap may be changed at any time.
