PegColor


The PegColor structure is used to pass color information to the PegScreen drawing functions. PegColor contains foreground, background, and drawing flag information. 

The PegColor structure is defined as: 


struct PegColor
{
   PegColor (COLORVAL fore, COLORVAL back = PCLR_DIALOG, UCHAR Flags = CF_NONE)
   {
      uForeground = fore;
      uBackground = back;
      uFlags = Flags;
   }

   PegColor()
   {
      uForeground = uBackground = BLACK;
      uFlags = CF_NONE;
   }

   void Set(COLORVAL fore, COLORVAL back = BLACK, UCHAR Flags = CF_NONE)
   {
      uForeground = fore;
      uBackground = back;
      uFlags = Flags;
   }

   COLORVAL uForeground;
   COLORVAL uBackground;
   UCHAR uFlags;
};
The color Flags are:

CF_NONE- This flag is used when the default operation is desired.

CF_FILL- This flag is used to fill rectangles, and polygons, and text. For text functions, CF_FILL will cause the text background area to be filled with the background color, while CF_NONE will cause only the text foreground to be drawn.
