PegHMIThing

Overview

PegHMIThing is the base class for all of the HMI gadgets in the PEG library. It provides a small layer of functionality over and above PegThing. PegHMIThing is typically not instantiated itself. Custom objects are derived from PegHMIThing to create new HMI gadgets.

Style Flags

PegHMIThing supports the standard frame styles FF_NONE, FF_THIN, FF_THICK, FF_RAISED, and FF_RECESSED.

Signals
PegHMIThing does not support any signals.

Derivation
PegHMIThing is derived from PegThing.
 

Constructors:

PegHMIThing(PegRect &Rect, WORD wStyle = FF_NONE, WORD wExStyle = 0)
The constructor takes a PegRect reference and a typical style flag (which it defaults to FF_NONE). It also takes an extended style flag which is used in all of the HMI gadgets to determine the look and feel of the gadget. It is important to remember that this extended flag in no way interferes with the regular style flags in PegThing derived objects.

Public Functions:

WORD GetExStyle(void) const
This function allows for the retrieval of the protected mwExStyle member which holds the extended style definition.

void SetExStyle(WORD wExStyle)
This function sets the mwExStyle member variable to define the extended style of the object.
