PegLinearScale

Overview

PegLinearScale is an output HMI gadget that facilitates a graphical representation of any given value in relation to a minimum and maximum value. This particular scale is the output equivalent of PegSlider.

See Also

PegScale

PegLinearBitmapScale

Style Flags

PegLinearScale supports the standard PegThing styles for borders: FF_NONE, FF_THIN, FF_RECESSED and FF_RAISED. See the PegScale Styles section for a detailed list of additional styles.

Signals

PegLinearScale does not send any signals.

Derivation

PegLinearScale is derived from PegScale.


Constructors:

PegLinearScale(PegRect &Rect, LONG lMinValue, LONG lMaxValue, LONG lTicFreq = 0, WORD wStyle = SS_STANDARDSTYLE | FF_RAISED)
The constructor takes a reference to a PegRect to determine its size, a typical wStyle WORD, a minimum value and a maximum value. The last parameter, lTicFreq, determines the frequency at which tick marks are drawn on the scale. For instance, if this is set to 100, and the minimum value is set to 0 and the maximum value is set to 1000, then PegLinearScale will draw a tick mark at every 100 data value increments starting with 0 and continuing up to 1000, making a total of 11 tick marks. It is important to remember that this value is in the context of the data value, not in screen pixels.

PegLinearScale(SIGNED iLeft, SIGNED iTop, LONG lMinValue, LONG lMaxValue, LONG lTicFreq = 0, WORD wStyle = SS_STANDARDSTYLE | FF_RAISED)
This constructor takes SIGNED values for the left and top pixel designations of the object. By default, this will construct a PegLinearScale object with a widht of 100 pixels and a height of 150 pixels. The remaining parameters are the same as the constructor above.

Public Functions:

virtual void Draw(void)
PegLinearScale overrides the Draw() function to draw the scale and the tick marks, if any.

LONG GetTicFreq(void) const
This inline method returns the current tick mark frequency.

void SetTicFreq(LONG lFreq)
This method allows for the setting of the interval at which the tick marks will be drawn.
