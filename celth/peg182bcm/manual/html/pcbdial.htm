PegCircularBitmapDial

Overview

PegCircularBitmapDial behaves exactly the same as PegCircularDial, from which it derives. The differences are the user may specify a background bitmap over which the dial needle will be drawn as well as a bitmap that will be used to draw the needle anchor at the center of rotation. This allows for very custom looking finite dials.

An added feature of PegCircularBitmapDial is the ability to specify a center of rotation that is not necessarily the center of the bounding rectangle for the dial. Typically, the needle of the dial would originate at the center of the bounding rectangle. PegCircularBitmapDial allows for the needle to originate from anywhere inside its bounding rectangle. This further adds to the custom possibilities when designing your dials.

PegDial

PegCircularDial

Style Flags

PegCircularBitmapDial supports the styles described in PegDial. PegCircularBitmapDial does not support any of the frame styles because it uses the background bitmap for drawing in the client area of the dial.

Signals

PegCircularBitmapDial does not send any signals.

Derivation

PegCircularBitmapDial is derived from PegCircularDial.


Constructors:

PegCircularBitmapDial(PegRect &Rect, SIGNED iRefAngle, LONG lValPerRev, LONG lMinValue, LONG lMaxValue, PegBitmap *pBackground, PegBitmap* pAnchorBitmap, WORD wStyle)
This constructor takes a reference to a PegRect to determine its size, a SIGNED value for the reference angle, a LONG value for the value per revolution, a minimum and maximum value, a pointer to a bitmap that will be used for drawing the background, a pointer to a bitmap that will be used to draw the needle anchor, and a style WORD.

PegCircularBitmapDial(SIGNED iLeft, SIGNED iTop, SIGNED iMinAngle, SIGNED iMaxAngle, LONG lMinValue, LONG lMaxValue, PegBitmap* pBackground, PegBitmap* pAnchorBitmap, WORD wStyle)
This constructor takes SIGNED values to the left and top positions of the dial. If a background bitmap is specified, the object is sized to the height and width of the bitmap. The remaining parameters are the same as the above constructor.

Public Functions:

virtual void Draw(void)
PegCircularBitmapDial overrides the Draw() function to draw the background bitmap and needle.

PegBitmap* GetAnchorBitmap(void) const
This method returns a PegBitmap pointer to the anchor bitmap.

PegBitmap* GetBkgBitmap(void) const
This method returns a PegBitmap pointer to the background bitmap.

WORD GetCORX(void) const
WORD GetCORY(void) const
These inline functions return the respective values of the x and y center of rotation.

void SetAnchorBitmap(PegBitmap* pBmp)
This method set the anchor bitmap to point to pBmp.

void SetBkgBitmap(PegBitmap* pBmp)
This method set the background bitmap to point to pBmp.

void SetCOR(WORD CORX, WORD CORY, BOOL bRedraw = FALSE)
This allows for the setting of the x and y values of the center of rotation (the origin of the needle) of the dial.

void UseTrueCOR(BOOL bUse)
This method is short hand for setting the style flags DS_RECTCOR and DS_USERCOR to on or off, depending on the value of bUse.
