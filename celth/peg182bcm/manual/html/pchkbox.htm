PegCheckBox

Overview

PegCheckBox is a PegButton class that provides a toggle operation. Any number of PegCheckBox objects with a common parent can be selected.

 

See Also

PegButton

PegTextButton

PegRadioButton

PegBitmapButton

Style Flags

PegCheckBox supports the BF_SELECTED and AF_ENABLED styles.

Signals
PegCheckBox sends the PSF_CHECK_ON signal when selected, and the PSF_CHECK_OFF signal when de-selected.

Derivation
PegCheckBox is derived from PegButton.
 
 

Constructors:

PegCheckBox(PegRect &Rect, const PEGCHAR *text, WORD wId = 0, WORD Style = AF_ENABLED)
PegCheckBox(SIGNED iLeft, SIGNED iTop, const PEGCHAR *text, WORD wId = 0, WORD wStyle = AF_ENABLED)
The first constructor creates a PegCheckBox with a user defined size. The second constructor allows the button to automatically size itself relative to the size of the textbox string.

Public Functions:

virtual void Draw(void)
PegCheckBox overrides the Draw() function to draw the checkbox appearance.

virtual BOOL IsChecked(void)
Returns TRUE if the checkbox is selected ON, else FALSE. 

virtual SIGNED Message(const PegMessage &Mesg)
PegCheckBox catches the PM_LBUTTONDOWN and PM_LBUTTONUP messages.

virtual void SetSelected(BOOL bSelected = TRUE)
This inline function can be called at any time to toggle the checkbox on or off under program control.
